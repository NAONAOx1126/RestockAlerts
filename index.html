再入荷通知プラグイン

メールアドレスを登録後、商品が購入可能になった際にメールで通知するEC-CUBE用プラグインです。

Download this project as a .zip file Download this project as a tar.gz file

再入荷通知プラグインのインストール

再入荷通知プラグインはこのページよりtar.gz形式でダウンロードし、そのファイルをEC-CUBE管理画面のプラグイン管理よりアップロードすることでインストールできます。 現在はアップデート処理の正常動作が確認できていないため、お手数ですが、バージョンアップの際は一度削除した上で再インストールを行なって下さい。

インストール後はメール設定に再入荷通知メールの項目が追加されますので、このメールを編集してください。

プラグインの動作

プラグインを有効にすると、在庫0の商品のカートに入れるボタンがメールフォームになります。

そこでメールアドレスを入力すると、その商品に対してメールアドレスが登録されます。

メールアドレス登録後、その商品の数量が1以上、もしくは在庫無制限となった場合にメールが送信されます。

該当商品が複数存在した場合には、1つのメールに対して複数の商品を記載して送信します。
